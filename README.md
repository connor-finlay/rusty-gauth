# Rusty-GAUTH

```shell
A command line tool to decode Google Authenticator export data

Usage: rusty-gauth [OPTIONS]

Options:
  -p, --path <img file>
          Path to image containing QR-code

  -l, --link <link>
          Migration link decoded from QR-code, starts with: otpauth-migration://offline?data=

  -h, --help
          Print help
```
